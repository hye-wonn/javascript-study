# 1. 주석

## 1-1. 단일 행 주석

-   `//` 기호 사용하여 한 줄 주석 작성

```js
let temperature = 26; // 현재 온도 (도)
let weather = 'sunny'; // 현재 날씨
console.log(temperature, weather); // 26 'sunny'
```

## 1-2. 다중 행 주석

-   `/* */` 기호로 여러 줄 주석 작성

```js
/*
이 코드는 오늘의 날씨 정보를 저장합니다.
temperature는 현재 온도를,
weather는 날씨 상태를 나타냅니다.
*/
let temperature = 26;
let weather = 'sunny';
```
